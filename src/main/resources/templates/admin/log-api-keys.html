Quick Start

Send a log entry with:

curl -X POST /api/logs \
  -H "Content-Type: application/json" \
  -H "X-Log-Api-Key: <your-key>" \
  -d '{"level":"ERROR","service":"api","message":"Something failed"}'

For batch ingestion use POST /api/logs/batch with {"logs":[...]}
